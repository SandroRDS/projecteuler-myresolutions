Introdução

O que é o Project Euler ?

O Project Euler é uma plataforma incrível que abriga um vasto repositório de problemas matemáticos com diversos níveis de dificuldade distintos. Foi criado em 2001 por Colin Hughes e tem como principal alvo programadores, possibilitando praticar e aprimorar sua Lógica de Programação e exercitar seus conceitos de Algoritmos. Porém, muito além de alcançar apenas programadores e desenvolvedores, o objetivo principal do projeto, segundo o próprio criador:

"O Project Euler existe para encorajar, desafiar e desenvolver as habilidades e o prazer de qualquer pessoa interessada no fascinante mundo da matemática."
Project Euler - Site Oficial

Qual o objetivo deste site ?

A criação deste site é um projeto pessoal totalmente voltado a fins didáticos, tendo como objetivo criar um repositório interativo com minhas resoluções, detalhadamente explicadas, de todos os problemas do Project Euler. Sendo assim, é recomendado que consulte as resoluções apenas se você já resolveu o problema em questão e deseja ver uma outra forma de solucioná-lo ou caso tenha se deparado com uma barreira epistemológica que não conseguiu superar e gostaria de expandir seus horizontes no pensamento da Lógica de Programação e Algoritmos.

Vale ressaltar que todas as resoluções aqui presentes são de minha autoria, porém, todos os enunciados são de total crédito dos criadores do Project Euler. Além disso, os algoritmos e as explicações aqui apresentadas são apenas uma forma (de milhares) de encontrar cada resultado, sendo assim, não são uma verdade absoluta e muito menos o definitivo melhor algoritmo para solucionar o problema em questão.
